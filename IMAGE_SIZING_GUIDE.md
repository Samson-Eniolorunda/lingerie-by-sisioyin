# Image Sizing Guide — Lingeries by Sisioyin

Use this guide when uploading images to ensure the best visual quality across all devices.

---

## Hero Image (Homepage)

The hero image appears as the main visual on the homepage, displayed in a 3:4 portrait frame at up to 450 px wide.

| Property          | Value          |
| ----------------- | -------------- |
| **Aspect Ratio**  | 3:4 (portrait) |
| **Recommended**   | 900 × 1200 px  |
| **Minimum**       | 600 × 800 px   |
| **Format**        | WebP or JPEG   |
| **Max File Size** | ~400 KB        |

> Use a high-quality model or lifestyle shot. The image fills the frame using `object-fit: cover`, so keep the subject centered and avoid important details near edges. Rounded corners are applied automatically.

---

## Product Card Images (Shop & New Arrivals)

Product images are shared across the Shop page and New Arrivals section — upload once and they appear in both.

| Property          | Value          |
| ----------------- | -------------- |
| **Aspect Ratio**  | 4:5 (portrait) |
| **Recommended**   | 800 × 1000 px  |
| **Minimum**       | 600 × 750 px   |
| **Format**        | WebP or JPEG   |
| **Max File Size** | ~200 KB        |

> The product should be **centered** in the frame. Images are cropped to fill the card, so keep the subject away from edges.

## General Tips

- **Format:** Use **WebP** for the smallest file size with great quality. JPEG is also fine.
- **Centering:** Always center the product/model in the photo. The site uses `object-fit: cover`, meaning edges may be cropped on different screen sizes.
- **Background:** Clean, uncluttered backgrounds make products stand out.
- **Consistency:** Use the same aspect ratio for all products in a section for a uniform grid.
- **Compression:** Compress images before uploading. Tools like [Squoosh](https://squoosh.app) or [TinyPNG](https://tinypng.com) help reduce file size without visible quality loss.
